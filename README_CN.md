# 南光尺寸裁剪插件

![南光工具](icons/nanguang.png)

一个专为ComfyUI设计的图像裁剪插件，提供九种裁剪类型，支持精确的尺寸控制和位置调整。

## 🚀 快速开始

### 安装方法

1. 将 `ComfyUI-Nanguang-Size` 文件夹复制到 `ComfyUI/custom_nodes/` 目录
2. 重启 ComfyUI
3. 在节点菜单中找到 "南光尺寸" 类别

### 使用步骤

1. 加载图像 (LoadImage节点)
2. 添加 "南光图像裁剪" 节点
3. 连接图像到节点
4. 设置裁剪参数：
   - 裁剪开关：开启/关闭裁剪功能
   - 裁剪类型：选择九种裁剪位置之一
   - 裁剪宽度：设置裁剪区域宽度
   - 裁剪高度：设置裁剪区域高度
   - X偏移：水平方向偏移调整
   - Y偏移：垂直方向偏移调整
5. 运行工作流查看结果

## 🎯 功能特性

### 九种裁剪类型
- **左顶** - 左上角裁剪
- **顶中** - 顶部中间裁剪
- **右顶** - 右上角裁剪
- **左中** - 左侧中间裁剪
- **居中** - 正中间裁剪（默认）
- **右中** - 右侧中间裁剪
- **左底** - 左下角裁剪
- **底中** - 底部中间裁剪
- **右底** - 右下角裁剪

### 精确控制
- 可设置任意宽度和高度
- 支持像素级偏移调整
- 实时验证裁剪尺寸
- 详细处理信息反馈

### 智能保护
- 自动检测图像尺寸
- 防止裁剪区域超出边界
- 错误提示和友好报错
